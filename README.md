# Truyện Thanh Xuân Kịch Tính: Khánh và Ngọc

Một ứng dụng Flutter nhỏ kể về câu chuyện thanh xuân cấp 3 đầy mâu thuẫn, kịch tính và những ngã rẽ bất ngờ của hai nhân vật Khánh và Ngọc.

## Tính năng
- Đọc truyện ngắn với giao diện thẻ (card) trực quan.
- Chuyển tiếp các phần của câu chuyện dễ dàng.
- Giao diện người dùng tối giản và dễ đọc trên mọi thiết bị.

## Cài đặt và Chạy ứng dụng
1. Đảm bảo bạn đã cài đặt [Flutter](https://flutter.dev/docs/get-started/install).
2. Clone repository này.
3. Chạy `flutter pub get` để tải các dependencies.
4. Chạy `flutter run` để khởi động ứng dụng.
